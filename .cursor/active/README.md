# Active Work Folder

Store execution artifacts for a specific feature.

Path format:
- `.cursor/active/<feature-slug>/plan.md`
- `.cursor/active/<feature-slug>/progress.md`
- `.cursor/active/<feature-slug>/result.md`

Templates:
- `.cursor/active/templates/plan-template.md`
- `.cursor/active/templates/progress-template.md`
- `.cursor/active/templates/result-template.md`
