# Feature: любимые карточки (избранное)

## Scope

- Публичное избрание своих карточек: флаг в БД, PATCH, счётчик и список `favorites_only` в API.
- UI сердечка и блок «Любимое» в приложении — по отдельному объёму (см. `docs/features/movie-card-favorites.md`).

## Acceptance

- Владелец может включать/выключать избранное через PATCH.
- Ответы деталки, ленты и списка карточек содержат `is_favorite`.
- Профиль отдаёт `favorites_count`.
- Список избранных сортируется по последней отметке.
