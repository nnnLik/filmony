# Feature: frontend-subscriptions-ui

## Scope
- Под именем пользователя показывать три счётчика: `подписчики`, `подписки`, `фильмы`.
- Клик по `подписчики`/`подписки` открывает отдельную страницу подписок.
- Отдельная страница подписок доступна для своего и чужого профиля.
- На странице подписок поддержать `Подписаться/Отписаться`.
- Для своего профиля на странице подписок показать блок шаринга публичной ссылки.

## UX Notes
- Навигация на страницу подписок по аналогии со страницей `Редактирование`.
- Для списков используется переключатель `Подписки / Подписчики`.
- Для счётчиков используются backend-поля `followers_count` / `following_count`.

## API Dependencies
- `POST /api/users/{user_id}/subscriptions`
- `DELETE /api/users/{user_id}/subscriptions`
- `GET /api/users/{user_id}/subscriptions?type=followers|following|both`
- Расширенные поля профиля: `followers_count`, `following_count`
