# Feature: feed-posts

## Scope

Backend MVP: модель `FeedPost`, миграция, создание и чтение по id, тесты. Без фронтенда и без слияния с лентой карточек.

## Acceptance

- `POST /api/feed-posts` создаёт пост при валидных данных.
- `GET /api/feed-posts/{id}` возвращает пост или 404.
- Публикация из своего комментария копирует текст и привязку к карточке; чужой комментарий — 403.
