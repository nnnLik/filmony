# Feature: детали статистики профиля (`profile-stats-details`)

## Цель
Показать в профиле пользователя расширенную статистику по карточкам фильмов:

- Всего фильмов
- Средняя оценка
- Распределение оценок
- Распределение по годам выпуска
- Популярные теги
- С кем смотрю
- Настроение после просмотра
- Топ фильмов (5)
- Худшие фильмы (5)

## Scope
- Backend: единый endpoint агрегатов для пользователя.
- Frontend: использование endpoint в `/profile` и `/u/:userId` на вкладке «Статистика».

## Acceptance Criteria
1. `GET /api/users/{user_id}/stats` возвращает агрегированную статистику по карточкам.
2. Endpoint требует авторизации и отдает `404` для несуществующего пользователя.
3. На фронтенде статистика отображается в моем профиле и публичном профиле.
4. Для отсутствующих данных корректно отображаются empty-состояния.
