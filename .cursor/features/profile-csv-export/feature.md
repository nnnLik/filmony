# Экспорт карточек в CSV через Telegram

## Scope

- На экране «Мой профиль» доступна кнопка экспорта.
- По нажатию бэкенд собирает **все** карточки пользователя в CSV (UTF-8 с BOM) и отправляет файл в личку с ботом через `sendDocument`.

## Acceptance criteria

- `POST /api/me/cards/export-csv` под авторизацией; ответ `{ "status": "sent" }` при успехе.
- Ошибки чата с ботом — `422` с `telegram_chat_unavailable` и `bot_username` (как у ping).
- Фронт: `IconButton` с иконкой загрузки в шапке профиля; сообщения об успехе и ошибках в духе страницы редактирования.
- Pytest покрывает успех (мок Telegram), 422, 502, пустой профиль.
